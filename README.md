# vfox-redis-exporter

[![CI](https://github.com/jbox-web/vfox-redis-exporter/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-redis-exporter/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [redis_exporter](https://github.com/oliver006/redis_exporter)

## Install

```
mise plugin install redis-exporter https://github.com/jbox-web/vfox-redis-exporter.git
```
